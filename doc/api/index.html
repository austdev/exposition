Exposition API API Reference

Describe the API used by the HTML5/JavaScript to access images and album resources.

Schemes: https, http
Version: 0.5

Paths

Read configuration

GET /config
200 OK
Configuration

WebSite configuration

500 Internal Server Error

Internal Error

Response Example (200 OK) 
{
  "gallery_name": "string",
  "info": "string",
  "is_download_allowed": "boolean",
  "photo_sizes": "array",
  "thumbnail_sizes": "array",
  "version": "string"
}

Download image

GET /download/{path}

The image is downloaded at full resolution.

path

Path of the Photo or Album to download

type
string (path)
in
path
200 OK

A zip archive containing all the albums and photos

500 Internal Server Error

Any Error

Get Image

GET /image/{size}/{path}

Obtain image at given resolution. This endpoint is used for both thumbnails and lower tier resolution images.

size

Size of the image. 0 means 'full size'

type
number (int32) , { x ∈ ℝ | x ≥ 0 }
in
path
path

Path of the photo to download. This must be the path of a 'photo' item not an 'album'

type
string (path)
in
path
200 OK

The image representing a photo item

500 Internal Server Error

Any Error

Get Item

GET /item/{path}

Obtain information about the item, either an Album or a Photo.

path

Path of the album item

type
string (path)
in
path
200 OK
Item

The album item

500 Internal Server Error

Any Error

Response Example (200 OK) 
{
  "children": "array",
  "path": "string(path)",
  "title": "string",
  "type": "string"
}

Models

Configuration: object

version: string

Currently fixed to '0.5'

info: string

Currently fixed to 'Exposition 0.5 beta. (c) 2013-2016 Aymeric Barthe.'

gallery_name: string

Name of the Gallery. User facing

photo_sizes: number[]

Sizes of photos. Positive integers only. '0' means original size.

thumbnail_sizes: number[]

Sizes of thumbnails. Positive integers only. '0' means original size.

is_download_allowed: boolean

Whether the user should be allowed to download photos.

Item: object

type: string , x ∈ { photo , album }

Indicates if the item is an 'album' or a 'photo'

title: string

Description of the item, or filename

path: string (path)

Path of the item

children: object[]

Children items (only present for type 'Album'). The type of the elements is also 'Item', recursively.
